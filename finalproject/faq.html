Student Information Database Web System FAQ's

Who Can Access This Student Information Database?

Only The Administrator Can Access The Student Information Database

How To Access The Student Information System Register Form

Only The Registered Administrator Can Access The Student Information Database Using Given Username And Password

How To Use Student Information System Register Form

1. Go To Homepage And Login Using The Username and Password Given

2. Admin Will Be Redirected To Database Student Form

3. Admin Can Key In The New Student Data in The Form Provided

4. Admin Also Can View The Existing and New Student Data By Clicking The Navbar Button
